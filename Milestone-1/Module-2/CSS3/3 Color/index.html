In CSS Colors can be specified using

  1. Predifined Color names(Modern browsers support 140 named colors) 140
  2. RGB - rgb(red, green, blue)
  3. RGBA - rgba(red, green, blue, alpha)
  4. HEX - #ff0000
  5. HSL - hsl(heu, saturation, lightness)
  6. HSLA - hsla(heu, saturation, lightness, alpha)
